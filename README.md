1. Clone the repository:
   ```bash
   git clone https://github.com/miasmith81/mia-smith-luna.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mia-smith-luna
   ```
3. Open `index.html` in your browser or use a local server.
